Usage

primes [S] [M[:N]] [G]
S is the problem size (number of primes). M and N are a range of numbers of threads to be used. G is an optional grain size.
To run a short version of this example, e.g., for use with Intel® Threading Tools:
Build a debug version of the example (see the build directions).
Run it with a small problem size and the desired number of threads, e.g., primes 100000 4.
